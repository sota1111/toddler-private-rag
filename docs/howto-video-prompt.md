# 使い方の動画 生成プロンプト（動画生成AI向け）

このドキュメントは、保育園おたより管理アプリ **「おたよりナビ」** の使い方紹介動画を、
汎用の動画生成AI（Runway / Pika / Sora / Google Veo など）で作るための **プロンプト** です。
下の「■ プロンプト本文」をそのままコピーして動画生成AIに貼り付け、**提供画像1〜7**として
リポジトリ内の既存キャプチャ（`frontend/public/howto/*.png`）を添付してください。

- 動画は「アプリのスクリーンショット（静止画）を順に全画面表示し、下部に字幕を重ねる」
  スライドショー形式です。**画面（スクショ）以外の実写・人物・背景映像は生成しません。**
- 字幕はアプリ内「使い方」画面（`HowToPage`）の実文言（`howto.*`）をそのまま使い、
  アプリとの表記ズレを防いでいます。

## ■ プロンプト本文（ここからコピーして動画生成AIに貼り付け）

```
あなたはプロの動画エディターです。保育園の「おたより」管理アプリ「おたよりナビ」の
使い方を紹介するチュートリアル動画を作成してください。

# 全体方針
- 形式: 私が添付するアプリのスクリーンショット画像（提供画像1〜7）を1枚ずつ全画面で表示し、
  画面下部に字幕を重ねるスライドショー動画。
- 重要: 画面（スクリーンショット）以外の映像（実写・人物・オフィス・手・背景CG など）は
  一切生成しないこと。見せるのは私が渡すスクリーンショットのみ。
- 向き/比率: 縦向きスマホUI。アスペクト比 9:16（1080x1920 目安、元画像は 390x844 相当）。
- 各シーンの動き: 静止画に対してケンバーンズ効果（ゆっくりズームイン、または上下の軽いパン）
  のみ。派手なエフェクトや3D変形は使わない。
- シーン間トランジション: 0.3〜0.5秒のクロスフェード（またはシンプルなカット）。
- 全体尺: 60〜90秒程度。各説明シーンは字幕が無理なく読み切れる 8〜11秒。
- 字幕: 日本語。画面下部に配置。読みやすいサンセリフ体（例: Noto Sans JP）、
  白文字＋半透明の暗いバー（下地）で視認性を確保。1シーンにつき「タイトル字幕（1行・強調）」
  と「本文字幕（1〜3行）」を表示。タイトルは画面上部寄せでも可。
- 読み速度: 本文字幕はそのシーンの尺内で読み切れるよう表示。長い本文は前半/後半に分割して
  順に出してもよい。
- BGM: 明るく落ち着いた軽いインストゥルメンタルを控えめに。ナレーション音声は不要
  （字幕のみ）。ナレーションを付ける場合は字幕と同一テキストの日本語女性音声で穏やかに。

# シーン構成（順番厳守）

[シーン0 / オープニング 3〜4秒]
- 映像: シンプルな単色〜淡いグラデーション背景にアプリ名を表示（スクショなし）。
- タイトル字幕: おたよりナビ
- サブ字幕: 保育園のおたよりを、写真から自動でタスク化

[シーン1 / 8〜11秒] 提供画像1 = register.png（登録画面）
- タイトル字幕: 1. おたよりを写真から登録する
- 本文字幕: 「登録」画面でおたよりの写真を撮影・選択すると、文字を自動で読み取り、
  やること・持ち物・締め切りを自動で仮登録します。内容を確認して本登録します。

[シーン2 / 8〜11秒] 提供画像2 = tasks.png（やること一覧）
- タイトル字幕: 2. やることを確認する
- 本文字幕: 「やること」画面で、提出物・持ち物・締め切りを一覧で確認できます。
  完了したものはステータスを変更できます。

[シーン3 / 10〜12秒] 提供画像3 = agent-demo.png（締切の自動逆算）
- タイトル字幕: 3. 提出書類の締切を自動で逆算する
- 本文字幕: 就労証明書などの提出書類は、最終締切から逆算して
  「様式の入手 → 発行依頼 → 記入内容の確認 → 市町村へ提出」の準備タスクを、
  締切に間に合う日付付きで自動生成します。やること画面に順番に並ぶので、早めに動けます。
- 補足演出: 「(1/4)〜(4/4)」の並びが見えるよう、リスト部分をゆっくり上へパンしてもよい。

[シーン4 / 8〜10秒] 提供画像4 = schedule.png（カレンダー）
- タイトル字幕: 4. カレンダーで予定を見る
- 本文字幕: 「カレンダー」画面で、行事や締め切りを日付ごとに確認できます。

[シーン5 / 8〜10秒] 提供画像5 = board.png（掲示板 / ホーム）
- タイトル字幕: 5. 掲示板で今日・今週を把握する
- 本文字幕: 「掲示板」画面（ホーム）で、今日・明日・今週・来週の予定やリマインドを
  まとめて確認できます。

[シーン6 / 8〜10秒] 提供画像6 = ask.png（質問）
- タイトル字幕: 6. わからないことを質問する
- 本文字幕: 「質問」画面で、登録したおたよりの内容について自然な言葉で質問できます。
  回答には根拠となるおたよりが表示されます。

[シーン7 / 8〜10秒] 提供画像7 = settings.png（設定）
- タイトル字幕: 7. 設定
- 本文字幕: 「設定」画面で、言語・タイムゾーン・市町村・お子さまの登録や、
  全データ削除ができます。

[シーン8 / クロージング 3〜4秒]
- 映像: オープニングと同系の背景（スクショなし）。
- 字幕: おたよりナビで、保育園のおたより管理をもっとかんたんに。

# 出力
- 上記の順序・字幕・尺で1本の縦型動画（9:16）を書き出してください。
- スクリーンショットは変形・トリミングしすぎず、画面全体が見えるように配置してください。
```

## 補足

- **英語版が必要な場合**: 各シーンの字幕を、`frontend/src/i18n/messages.ts` の `howto.*`
  英語文言（`registerTitle`/`registerBody` … `settingsTitle`/`settingsBody`）に差し替えてください。
  字幕テキスト以外（構成・尺・トランジション指示）はそのまま流用できます。
- **画像の更新**: キャプチャは `frontend/e2e/capture-howto.spec.ts`（および
  `capture-agent-demo.spec.ts`）で再生成されます。画面変更後にこれらを再撮影すると、
  本プロンプトはパス参照のままで最新画像を使えます。
- 本プロンプトの字幕は `HowToPage.tsx` の使い方セクションと同一文言・同一順序です。
  アプリ側の文言を変えた場合は、本ファイルの該当字幕も合わせて更新してください。
